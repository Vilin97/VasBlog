A puzzle

The other day my friend Leo presented the following puzzle to me. Given a partition {On}n∈I\left\{ \mathcal O_n \right\}_{n\in I}{On​}n∈I​ of positive integers with On\mathcal O_nOn​ all infinite, find a condition that is equivalent to III finite. The motivation for the problem was juggling: each On\mathcal O_nOn​ is the "orbit" of the nthn^{\text{th}}nth ball, and we want to know when we are juggling finitely many balls.

Mathematical observations

Width and density

After playing around with some examples we came up with the definition of width of an infinite subset of Z+\mathbb Z_+Z+​: the width of O\mathcal OO denoted by w(O)w(\mathcal O)w(O) is the supremum of distances between consecutive elements. Intuitively, width is kind of the reciprocal of natural density. Recall that the natural density of O\mathcal OO is the limit d(O)=lim⁡n→∞a(n)/nd(\mathcal O) = \lim_{n\to \infty} a(n)/nd(O)=limn→∞​a(n)/n where a(n)a(n)a(n) is the number of elements of O\mathcal OO in {1,...,n}\left\{ 1,...,n \right\}{1,...,n}. Therefore 1/w(O)≤d(O)1/w(\mathcal O) \leq d(\mathcal O)1/w(O)≤d(O). Let's see why that is.

Given O={xi}1∞⊂Z+\mathcal O = \left\{ x_i \right\}_1^\infty \subset \mathbb Z_+O={xi​}1∞​⊂Z+​, let y0=x1y_0 = x_1y0​=x1​ and yi=xi+1−xiy_i = x_{i+1} - x_iyi​=xi+1​−xi​ so that yiy_iyi​'s are the "gaps" between consecutive terms of the sequence O\mathcal OO. Then w(O)=sup⁡{yi:i≥1}w(\mathcal O) = \sup \left\{ y_i: i\geq 1 \right\}w(O)=sup{yi​:i≥1}. On the other hand, for n=xin = x_in=xi​

a(n)/n=i/xibecause a(xi)=i=i∑j=1iyj≥i∑j=1iw(O)=1/w(O).\begin{aligned} a(n)/n &= i/x_i \text{\hspace{2cm} because $a(x_i) = i$}\\ &= \frac{i}{\sum_{j=1}^i y_j} \geq \frac{i}{\sum_{j=1}^i w(\mathcal O)} = 1/w(\mathcal O). \end{aligned}a(n)/n​=i/xi​because a(xi​)=i=∑j=1i​yj​i​≥∑j=1i​w(O)i​=1/w(O).​

So taking lim⁡i→∞\lim_i \to \inftylimi​→∞ we get d(O)≥1/w(O)d(\mathcal O) \geq 1/w(\mathcal O)d(O)≥1/w(O). Additionally, if all but finitely many gaps yiy_iyi​ are equal to w(O)w(\mathcal O)w(O), then d(O)d(\mathcal O)d(O) exists and equals 1/w(O)1/w(\mathcal O)1/w(O).

Taking lim inf⁡i→∞\liminf_{i\to\infty}liminfi→∞​ of (1) actually shows that even the lower density d‾(O)=lim inf⁡na(n)/n\underline{d}(\mathcal O) = \liminf_n a(n)/nd​(O)=liminfn​a(n)/n is at least 1/w(O)1/w(\mathcal O)1/w(O). The advantage of using the lower density is that it is defined for all subsets of natural numbers. Let's record this observation.

O1O_1O1​: for O⊂Z+\mathcal O \subset \mathbb Z_+O⊂Z+​, 1/w(O)≤d‾(O)1/w(\mathcal O) \leq \underline{d}(\mathcal O)1/w(O)≤d​(O), where 1/w(O)=01/w(\mathcal O)=01/w(O)=0 if w(O)=∞w(\mathcal O) = \inftyw(O)=∞.

Growth of w(On)w(\mathcal O_n)w(On​)

Here is another easy observation:

O2O_2O2​: If ∀n∈I,w(On)≤M\forall n \in I, w(\mathcal O_n) \leq M∀n∈I,w(On​)≤M for some fixed integer MMM, then III is finite.

Proof: If III is infinite, consider M+1M+1M+1 sets O1,...,OM+1\mathcal O_1,...,\mathcal O_{M+1}O1​,...,OM+1​ and an interval [a,a+M−1][a,a+M-1][a,a+M−1] where aaa is large enough that the smallest element of each On,n≤M+1\mathcal O_n, n\leq M+1On​,n≤M+1 is below aaa. Then all M+1M+1M+1 disjoint sets O1,...,OM+1\mathcal O_1,...,\mathcal O_{M+1}O1​,...,OM+1​ have at least one element in [a,a+M−1][a,a+M-1][a,a+M−1], which is impossible by pigeonhole principle.

So if w(On)=O(1)w(\mathcal O_n) = O(1)w(On​)=O(1), then III is finite – we are juggling finitely many balls. This leads to another conjecture: if ∀n∈I,w(On)≤Mn\forall n \in I, w(\mathcal O_n) \leq M_n∀n∈I,w(On​)≤Mn​, then III is finite (the only difference is that now MnM_nMn​ depends on nnn). This one is not true. Let the first set fill half of naturals, the second – half of the remaining half, etc:

O1=2Z+, O2=1+4Z+, O3=3+8Z+, ..., On=2n−1−1+2nZ+.\mathcal O_1 = 2\mathbb Z_+,\ \mathcal O_2 = 1 + 4\mathbb Z_+,\ \mathcal O_3 = 3+8\mathbb Z_+,\ ... ,\ \mathcal O_n = 2^{n-1}-1 + 2^n\mathbb Z_+.O1​=2Z+​, O2​=1+4Z+​, O3​=3+8Z+​, ..., On​=2n−1−1+2nZ+​.

Here w(On)=2nw(\mathcal O_n) = 2^nw(On​)=2n with III infinite.

So far we know that if w(On)=O(1)w(\mathcal O_n) = O(1)w(On​)=O(1), then III is finite, but w(On)=O(2n)w(\mathcal O_n) = O(2^n)w(On​)=O(2n) is not strong enough to guarantee that. A natural question is:

Question: is there an increasing function f(n)f(n)f(n) such that w(On)=O(f(n))w(\mathcal O_n) = O(f(n))w(On​)=O(f(n)) if and only if III is finite? Perhaps f(n)=n2f(n) = n^2f(n)=n2 or another polynomial?

We already saw that 1/w(O)≤d‾(O)1/w(\mathcal O) \leq \underline{d}(\mathcal O)1/w(O)≤d​(O). So it might be fruitful to think about the (lower) densities of On\mathcal O_nOn​. Intuitively, the sum of densities of a partition cannot be larger than 1 because each density is the "proportion" of the set in Z+\mathbb Z_+Z+​. Indeed, this is easy to formally state and prove:

O3O_3O3​: if sets {On}n=1∞\left\{ \mathcal O_n \right\}_{n=1}^\infty{On​}n=1∞​ are disjoint, then ∑n=1∞d‾(On)≤1\sum_{n=1}^\infty \underline{d}(\mathcal O_n) \leq 1∑n=1∞​d​(On​)≤1.

Proof: It suffices to prove that ∑n=1Nd‾(On)≤1\sum_{n=1}^N \underline{d}(\mathcal O_n) \leq 1∑n=1N​d​(On​)≤1 for all NNN so fix any NNN. We have

∑n=1Nd‾(On)=lim inf⁡i→∞∑n=1N∣On∩[i]∣i=lim inf⁡i→∞1i∑n=1N∣On∩[i]∣≤lim inf⁡i→∞1i×i=1 because On’s are disjoint.\begin{aligned} \sum_{n=1}^N \underline{d}(\mathcal O_n) &= \liminf_{i\to\infty} \sum_{n=1}^N \frac{|\mathcal O_n\cap [i]|}{i} = \liminf_{i\to\infty} \frac{1}{i}\sum_{n=1}^N |\mathcal O_n\cap [i]|\\ &\leq \liminf_{i\to\infty} \frac{1}{i}\times i = 1 \text{ \hspace{1cm} because $\mathcal O_n$'s are disjoint}. \end{aligned}n=1∑N​d​(On​)​=i→∞liminf​n=1∑N​i∣On​∩[i]∣​=i→∞liminf​i1​n=1∑N​∣On​∩[i]∣≤i→∞liminf​i1​×i=1 because On​’s are disjoint.​

Putting observations O1O_1O1​ and O3O_3O3​ together we obtain

O4O_4O4​: If III is infinite, then ∑n=1∞1/w(On)≤1\sum_{n=1}^\infty 1/w(\mathcal O_n) \leq 1∑n=1∞​1/w(On​)≤1. In particular, 1/w(On)1/w(\mathcal O_n)1/w(On​) is asymptotically smaller than 1/n1/n1/n. Formally, w(On)=ω(n)w(\mathcal O_n) = \omega(n)w(On​)=ω(n).

The contrapositive of the above is w(On)=O(n) ⟹ ∣I∣<∞w(\mathcal O_n) = O(n) \implies |I| < \inftyw(On​)=O(n)⟹∣I∣<∞. Is the converse true? For example, can we find an infinite partition {On}\left\{ \mathcal O_n \right\}{On​} of Z+\mathbb Z_+Z+​ with w(On)≤n2?w(\mathcal O_n) \leq n^2?w(On​)≤n2? In order to answer this question I wrote a program that explores the "greedy" algorithm of making successive sets On\mathcal O_nOn​.

Programming exploration

Greedy approach

We want to see if we can "fit" infinitely many disjoint sets {On}\left\{ \mathcal O_n \right\}{On​} in Z+\mathbb Z_+Z+​ with w(On)≤n2w(\mathcal O_n) \leq n^2w(On​)≤n2 (starting at n=2n=2n=2). The naive greedy algorithm for doing this is as follows. Let O2=4,8,12,16,etc\mathcal O_2 = 4,8,12,16,\text{etc}O2​=4,8,12,16,etc. We now want to let O3=9,18,27,36,etc.\mathcal O_3 = 9,18,27,36,\text{etc.}O3​=9,18,27,36,etc. but there is a collision at 363636. So change 363636 to 353535 and keep going at strides of 999. In general, place the next element of On\mathcal O_nOn​, go to the current element plus the width n2n^2n2. If this spot is empty, put it there, if not, try the spot one below, etc. If we are ever unable to find the next spot because on our search down we returned back to the current element, then this strategy fails. If no such failure occurs, then we succeed.

The aforementioned program successfully uses this strategy for 10810^8108 numbers, i.e. we fill the set {1,…,108}\left\{ 1,\dots,10^8 \right\}{1,…,108} with sets O2,…,O108\mathcal O_2,\dots,\mathcal O_{\sqrt{10^8}}O2​,…,O108​​ without any problems. Additionally letting the width of On\mathcal O_nOn​ be c⋅n1+ϵc \cdot n^{1+\epsilon}c⋅n1+ϵ for higher constants ccc and smaller values of ϵ>0\epsilon>0ϵ>0 suggests that this strategy is successful for w(On)=Θ(n1+ϵ)w(\mathcal O_n) = \Theta(n^{1+\epsilon})w(On​)=Θ(n1+ϵ) for ϵ>0\epsilon>0ϵ>0 arbitrarily small. So we have the conjecture
